### 1) `ROADMAP.md`
- `H-2e`, `M-1e`, `M-2d`의 고정 pass 수치에 **"완료 당시"** 라벨 추가
- `L-2b`, `L-2c`를 현재 구현 범위로 구체화

  - CRC-8 only
- `L-3a`를 실제 shipped behavior로 축소
  - `33–64 → uint64_t`
  - `>64 → UnsupportedFeature`
  - 배열 기반 fallback은 backlog
- Tracking source-of-truth 우선순위를 최신 `Reports/` 중심으로 재정렬하고 `ORACLE_RESULTS.md`를 시점상 stale 가능 스냅샷으로 명시

### 2) `README.md`
- `crc_counter_check` 설명에 현재 구현 상태 반영
  - inferred CRC/counter-like signal에 대한 fail-fast guard
  - explicit `crc_counter:` YAML metadata 지원
  - 현재 MVP 알고리즘 범위는 CRC-8 only
- Limitations 섹션을 현재 상태와 맞게 수정
  - automatic name-heuristic validation 미구현
  - explicit metadata 기반 generated handling 지원
- 하단 CRC/Counter note를 현행 구현 범위로 갱신

### 3) `README.ko.md`

  - fail-fast guard와 explicit metadata 기반 생성 검증을 분리 설명
  - `validate` / `passthrough` / `fail_fast`, CRC-8 only 범위 반영

### 4) `AGENTS.md`
